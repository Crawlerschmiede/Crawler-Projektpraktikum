# Crawler-Projektpraktikum

- download godot from https://godotengine.org/download/windows/ (c# version is recommended, https://dotnet.microsoft.com/download is also needed in this case)
- download and install https://aka.ms/vc14/vc_redist.x64.exe for the git addon to work
- make sure git is installed and in your PATH
- run `git lfs install` to enable git lfs
